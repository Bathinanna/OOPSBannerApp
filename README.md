Branch: feature/UC1-PrintOOPS
Goal: Introduce basic printing in Java using OOPS principles.
Description:

Creates a basic BannerPrinter class.

UC2 – Banner Print Statements
Branch: feature/UC2-BannerPrintStatements
Goal: Expand banner output using multiple print statements.
Description:

Prints banner top/bottom borders.
Prints message line by line.
No loops or arrays used yet.


UC3 – Banner String Join
Branch: feature/UC3-BannerStringJoin
Goal: Introduce String operations.
Description:

Uses String.join() or concatenation to assemble messages.
Makes banner content cleaner and more maintainable.


UC4 – Banner Array Loop
Branch: feature/UC4-BannerArrayLoop
Goal: Introduce arrays + loops.
Description:

Stores banner text lines in an array.
Uses for/foreach loops to print banner dynamically.
Reduces repetitive print statements.


UC5 – Banner Array INIT
Branch: feature/UC5-BannerArrayINIT
Goal: Showcase array initialization techniques.
Description:

Demonstrates inline vs dynamic array initialization.
Allows dynamic updates to banner text.

Branch: feature/UC6-BannerPattern
